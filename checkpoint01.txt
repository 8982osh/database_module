2) Explain when a database would be used. Explain when a text file would be used.

A database would be used when you need to track and access specific data types. If you need to search for specific data types, make changes, and limit who can access the database, then a text file not be used.

A text file would be used when you don’t need to organize information that would need to be repeatedly accessed and changed. Information in a text file could be a combination of different data types used together.  

3) Describe one difference between SQL and other programming languages.

SQL has commands that read much like the English language. These commands request information. The requests do not tell SQL what steps to take. Rather the requests state what information the user needs.  

4) In your own words, explain how the pieces of a database system fit together at a high level.

The database is an organization of tables, columns and rows, which allow data to be structured, organized and searched for speed.  

5) Explain the meaning of table, row, column, and value.

A table is the structure that holds all the data. The table’s columns
hold a specific data type. The data type has a specific value. The row lists all the information from each column that belong to a unit.
